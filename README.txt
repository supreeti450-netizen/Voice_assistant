Voice Assistant Project

This project is a basic voice assistant created using Python.
It uses speech recognition to take voice commands and responds
using text-to-speech.

Features:
- Responds to greetings
- Tells current time and date
- Searches the web based on voice input
- Simple and beginner-friendly

How to Run:
1. Install required libraries
2. Run voice_assistant.py
3. Speak commands through microphone
